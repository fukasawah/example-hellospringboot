Hello SpringBoot
==================




Build
------------------

```
./mvnw package
```


Run
------------------

```
java -jar hellospringboot-0.0.1-SNAPSHOT.jar
```

API
------------------

### `/memory?size=100`


|Param|Type| Description |
|----|-----|-----------------------------------|
|`size`| `int` | Allocate memory. Unit:MiB. default: `1`|
